**direction + force + area <------>haptic**
- 其中direction 和 force 都是由CoTracker获得，area为我们的预测接触patch
- 可以对应到触觉图中的 shape（反映接触区域的物理3D形状），还有color（反应为力越大的地方颜色深度应该更深）
- shape 与 area的对应关系很明显，但是direction + force 与 color 的对应关系就没那么明显了
- 我们需要尝试找出direction + force 与 color 之间的映射关系，可能是简单的线性，也可能是非线性，或者是用神经网络
