VGGT:Visual Geometry Grounded Transformer
你目前的问题是：

> 已经用 CoTracker 得到了传感器在 2D 图像中的点集、速度和方向；接下来如何利用当前帧和先前帧预测未来接触区域？

VGGT 的价值是把这个问题从 2D 推进到 3D：

CoTracker 2D tracks+VGGT depth / point maps / cameras→3D sensor motion→future 3D contact region\text{CoTracker 2D tracks} + \text{VGGT depth / point maps / cameras} \rightarrow \text{3D sensor motion} \rightarrow \text{future 3D contact region}CoTracker 2D tracks+VGGT depth / point maps / cameras→3D sensor motion→future 3D contact region

这样可以解决 2D 方法的几个问题：

- 2D 运动方向不等于真实接近方向；
- 2D 距离不等于真实空间距离；
- 图像中看似接近，3D 中可能不接近；
- 接触力与法向方向有关，必须有 3D 几何；
- 接触区域在 2D 中可能受透视影响变形。
